🎛️ 按钮参考

本插件在 Ribbon 自定义选项卡 DeepSeek 下提供 13 个按钮，覆盖 6 个分组。所有写入操作都是两步模式（AI 建议 → 用户确认 Apply）。

分组 1：AI 助手

按钮说明
📊 Excel 上下文抓取当前选区（地址 + 表头 + 样例），写入对话上下文。
🧹 PII 脱敏扫描选区里的身份证/手机/邮箱/银行卡号，AI 给出脱敏建议。
🔧 代码生成把选中的公式转换成 VBA 或 Office Scripts。
📈 图表推荐基于选区数据推荐 3 种最合适的图表。

分组 2：公式与数据

按钮说明
📝 公式解释把选中的公式逐段拆解，说明每部分作用。
🔍 公式诊断扫描选区里的公式错误（#REF! #DIV/0! 等），AI 给出修复建议。
🛠 修复公式一键把诊断建议应用到选中的错误公式上。

分组 3：可视化与代码

按钮说明
📐 代码格式化对选中的 VBA / 公式进行格式化重排。

分组 4：数据保护

按钮说明
🔒 脱敏写入把选中的敏感值替换为脱敏值并写入原单元格。

分组 5：AI 分析（Phase 4）

按钮说明
🟩 相关性矩阵计算所有数值列的 Pearson 相关性，写入 NxN 热力图（带条件格式）。
🟥 异常值检测用 z-score 方法扫描数值列，高亮异常单元格。
🧮 AI 透视表基于数据特征，AI 推荐 rows / columns / values 布局，确认后一键创建 PivotTable。

分组 6：报表与数据（Phase 4）

按钮说明
📝 快速报告对选中数据生成结构化报告（标题/摘要/章节/建议），写入新工作表。
🏷️ 列类型推断推断每列类型（数字/日期/分类/货币/百分比），Apply 后加表头样式 + 命名范围 + 数字格式。

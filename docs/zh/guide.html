🚀 用户指南

本指南面向第一次使用本插件的用户。读完即可上手所有日常场景。

1. 安装与启动

  1. 在 Office 365 / LTSC Standard / Mac 2024 中打开 Excel。
  2. 通过 插入 → 我的加载项 加载 manifest.xml。
  3. 任务窗格顶部会出现 DeepSeek 选项卡。
  4. 点 ⚙ 图标打开设置面板，填入 API Key 和模型端点。

2. 第一次对话

  1. 在任务窗格的输入框里键入问题，例如 "帮我总结这张销售表"。
  2. 可选：先在 Excel 里选中一段数据，再点 📊 选区 按钮把数据拼到 prompt 后面。
  3. 点 发送。AI 流式回复会逐字显示。
  4. 点 📌 插入 可把 AI 回复直接写回当前单元格。

3. 快捷键

快捷键动作
⌘ K / Ctrl K打开命令面板
⌘ N新建会话
⌘ H打开/关闭历史侧栏
⌘ B知识库
⌘ ⇧ S分享会话
⌘ D用量看板
⌘ /快捷键说明
Esc关闭弹窗/面板

4. Excel 上下文抓取

AI 对话可以引用当前选中的数据。具体做法：

  • 在 Excel 里 选中一段连续区域（含表头）。
  • 点 Ribbon 上的 📊 Excel 上下文 按钮，或点任务窗格的 📊 选区。
  • 系统会把选中区域的地址、表头、前 50 行样例拼到 prompt 后面。
💡 小贴士：样例数据越多，AI 推断越准。但超过 200 行后会截断，以免 token 超限。

5. 系统提示词（高级）

在设置面板的 系统提示词 字段，你可以自定义 AI 角色。例如：

你是一名资深财务分析师，回答必须严谨，给出可验证的 Excel 公式或 VBA 代码。

系统提示词对所有会话生效。

6. 历史会话

  • 最多保留 50 个会话，超过后自动删除最早的。
  • 支持按标题搜索（⌘ H 打开侧栏后顶部输入）。
  • 可重命名、导出 Markdown、批量删除。

7. 常见误区

  • ❌ 不要把所有数据一次发给 AI —— 用 选区 控制范围。
  • ❌ 不要把敏感数据（身份证号、手机号）粘贴到 prompt 里 —— 用 PII 脱敏 按钮先扫一遍。
  • ✅ 多轮对话时引用上一轮的表格/字段，AI 会自动结合上下文。

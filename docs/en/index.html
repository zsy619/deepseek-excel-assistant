About the add-in

DeepSeek Excel Assistant is an Office Add-in that brings DeepSeek LLM conversation into Excel's task pane. Streaming output, Markdown rendering, code highlighting, and user-cancellable at any time.

Core capabilities

  • Chat — in-pane conversation with full context retention.
  • Excel context — one-click capture of the active selection into the prompt.
  • Insert to cell — pipe AI replies back into the active cell.
  • Formula diagnostics — scan a range for errors and get AI fixes.
  • Ribbon enhancements — 13 buttons across 6 groups (formula gen, charts, PII masking, reports, correlation, outliers, pivots, column types).
  • History — up to 50 sessions, searchable, rename, Markdown export.

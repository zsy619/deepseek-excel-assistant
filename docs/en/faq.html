❓ FAQ

Account & config

Q: Where do I put my API key?

Click ⚙ Settings in the top-right of the pane; API Key is the first field. The key is stored only locally (localStorage) and never uploaded to any third party.

Q: Which models / endpoints are supported?

The default endpoint is DeepSeek official: https://api.deepseek.com/v1/chat/completions. Any OpenAI-compatible proxy works — set the endpoint URL to your gateway (e.g. Azure OpenAI, local Ollama).

Q: Can I use GPT-4?

Yes. Point the endpoint to OpenAI / Azure OpenAI and set model to gpt-4o, gpt-4-turbo, etc.

Q: How do I write a good system prompt?

Use role + behavior constraints:

You are a senior financial analyst.
- Reply in clear Chinese.
- Produce verifiable Excel formulas.
- Do not fabricate data.

Features

Q: Can I undo an AI write?

Yes for cell writes (Office ⌘ Z). Batch writes (pivot / report) must be removed manually.

Q: How many formulas can diagnostics handle?

Default 50 per scan (FORMULA_DIAGNOSIS_LIMIT). Larger ranges get a "shrink the selection" prompt.

Q: PivotTable creation fails — what now?

  • Selection not rectangular (merged cells, gaps).
  • Mixed types in one column.
  • Office < 2016 (no PivotTable API).

Q: Column type confidence is low?

Below 50% AI marks it unknown. Try cleaning data first, or providing ≥ 20 sample rows.

Limits

Q: Token limits?

Sample data: up to 50 rows × 20 columns. Single message: estimated > 8K gets truncated with a warning.

Q: Streaming stutters?

Check network (VPN / corporate proxy). Lowering Temperature to 0.3 produces shorter, more deterministic output.

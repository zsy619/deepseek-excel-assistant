🎛️ Button Reference

The add-in exposes 13 buttons under a custom Ribbon tab DeepSeek, grouped into 6. Every write operation is two-step (AI proposes → user confirms Apply).

Group 1: AI Assistant

ButtonWhat it does
📊 Excel ContextCaptures the active selection (address + headers + samples) into the conversation.
🧹 PII MaskScans the selection for IDs / phones / emails / cards and proposes masks.
🔧 Code GenConverts the selected formula into VBA or Office Scripts.
📈 Chart SuggestPicks 3 most fitting chart types for the selected data.

Group 2: Formula & Data

ButtonWhat it does
📝 Explain FormulaDecomposes the selected formula piece by piece.
🔍 DiagnoseScans selected formulas for errors and proposes fixes.
🛠 Apply FixOne-click apply of the proposed fix in-place.

Group 3: Visualization & Code

ButtonWhat it does
📐 Format CodePretty-prints selected VBA / formula text.

Group 4: Data Protection

ButtonWhat it does
🔒 Mask In PlaceReplaces sensitive values in the selection with masked versions.

Group 5: AI Analysis (Phase 4)

ButtonWhat it does
🟩 Correlation MatrixPearson correlation of all numeric columns → NxN heatmap with conditional formatting.
🟥 Outlier Detectionz-score scan, highlights outlier cells.
🧮 AI PivotAI proposes rows/columns/values, Apply creates a PivotTable on a new sheet.

Group 6: Report & Data (Phase 4)

ButtonWhat it does
📝 Quick ReportGenerates a structured report (title / summary / sections / recommendations) and writes it to a new sheet.
🏷️ Column TypesInfers each column type (number / date / categorical / currency / percent) → Apply bolds headers, autofilters, adds named ranges, sets number format.

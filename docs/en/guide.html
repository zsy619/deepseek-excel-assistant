🚀 User Guide

For first-time users. By the end you should be able to cover every common workflow.

1. Install & launch

  1. Open Excel on Office 365 / LTSC Standard / Mac 2024.
  2. Insert → My Add-ins → load manifest.xml.
  3. The DeepSeek tab appears in the Ribbon.
  4. Click the ⚙ icon in the task pane and fill in your API Key + endpoint.

2. First conversation

  1. Type a question in the pane input, e.g. "summarize this sales sheet".
  2. Optional: select a range in Excel first, then click 📊 Selection to attach it.
  3. Hit Send. The stream will reveal the reply token by token.
  4. Click 📌 Insert on a reply to write it back to the active cell.

4. Excel context capture

  • Select a contiguous range (with headers) in Excel.
  • Click Ribbon 📊 Excel context or pane 📊 Selection.
  • The address, headers, first 50 sample rows are appended to the prompt.
💡 Tip: more samples → better inference. Beyond 200 rows it gets truncated to stay under token limits.

5. System prompt (advanced)

In the Settings panel, System prompt lets you define an AI persona, e.g.:

You are a senior financial analyst. Be rigorous; produce verifiable Excel formulas or VBA.

The system prompt applies to all sessions.

6. History

  • Up to 50 sessions retained; oldest auto-pruned.
  • Search by title (after opening the sidebar with ⌘ H).
  • Rename, export Markdown, batch delete.

7. Common pitfalls

  • ❌ Don't dump your entire dataset into one prompt — use Selection to scope.
  • ❌ Don't paste raw PII (IDs, phone numbers) — run PII Mask first.
  • ✅ In multi-turn chats, reference prior tables/columns; the model keeps context.

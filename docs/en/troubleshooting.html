🔧 Troubleshooting

1. Add-in not loading / blank pane

  • Validate manifest: office-addin-manifest validate manifest.xml
  • WEF scanner: requires Version="1.0.0.0" + AID + SourceLocation matching exactly.
  • Dev server: confirm https://localhost:3000/taskpane.html reachable.
  • Certs: office-addin-dev-certs install.

2. "Add-in error" dialog

Usually dev server down or port collision.

lsof -i :3000
kill -9 <PID>
npm start

3. AI request failures

401 Unauthorized

  • Check API key (no stray spaces).
  • Endpoint must match the provider.

429 Too Many Requests

  • Quota exhausted. Check provider dashboard.
  • Corporate rate limit — wait and retry.

500 / 502 / 503 / 504

  • Provider outage. Retry shortly.
  • Switch models (chat ↔ reasoner).

Network / CORS

  • Self-hosted proxy may block OPTIONS preflight.
  • Whitelist api.deepseek.com in corporate gateway.

4. Ribbon button does nothing

  • Verify FunctionFile points to https://localhost:3000/commands.html.
  • Open DevTools (F12) and check Console.
  • Confirm commands.js registered all ExecuteFunctions.

5. Write failures

"Cannot parse selection"

Selection contains merged or non-contiguous cells.

"Permission denied"

Workbook is read-only or protected — unprotect first.

"PivotTable creation failed"

Source not on the active sheet, or headers missing.

6. Logs

Press F12 in the task pane. Watch the Console:

  • DS: stream — streaming events
  • DS: apply — Apply phase
  • DS: error — errors

7. Full reset

localStorage.clear();
location.reload();
⚠️ This clears all API keys and history. Export Markdown first.
